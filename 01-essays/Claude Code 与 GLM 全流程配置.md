## 前置工作：NVM 安装与配置

### 1\. 下载与安装 NVM

-   **下载地址**：[点击自动下载 nvm-1.2.2-setup.zip](https://nvm.uihtm.com/nvm-1.2.2-setup.zip)
    
-   **安装步骤**：
    
    1.  下载得到 `nvm-1.2.2-setup.zip` 压缩包。
        
    2.  解压缩得到 `nvm-setup.exe` 并运行安装。
        
    3.  **重要**：安装成功后，请**关闭并重新打开** CMD 或 PowerShell（命令行），以确保环境变量生效。
        

### 2\. 验证安装

在命令行中执行：

```bash
nvm --version
```

若输出 `1.2.2`，代表下载并安装成功。

### 4\. 安装与切换版本

#### 第一步：安装指定版本

执行命令安装（示例安装 LTS 版本 `24.14.0`，具体版本号请替换为您需要的版本）：

```bash
nvm install 24.14.0
```

#### 第二步：使用该版本

安装完成后，执行以下命令切换到该版本：

```bash
nvm use 24.14.0
```

此时终端会提示 `Now using node v24.14.0...`，表示切换成功。

## 开发工具接入之 Claude Code

详情可查看[智谱 AI 开放平台](https://docs.bigmodel.cn/cn/coding-plan/tool/claude#claude-code)（推荐仔细阅读）

### 步骤一：安装 Claude Code

进入命令行界面，安装 Claude Code

```bash
npm install -g @anthropic-ai/claude-code
```

运行如下命令，查看安装结果，若显示版本号则表示安装成功

```bash
claude --version
```

### 步骤二：配置 GLM Coding Plan

进入命令行界面，执行如下运行 Coding Tool Helper

```bash
npx @z_ai/coding-helper
```

按照显示的交互页面操作即可（详细说明请参考 [Coding Tool Helper 文档](https://docs.bigmodel.cn/cn/coding-plan/extension/coding-tool-helper)）

![Description](https://cdn.bigmodel.cn/markdown/1764741445765coding-tool-helper.gif?attname=coding-tool-helper.gif)

### 步骤三：开始使用 Claude Code

配置完成后，进入一个您的代码工作目录，在终端中执行 `claude` 命令即可开始使用 **Claude Code**

> 进入一个目录，右键空白部分，选择`在终端中打开`，输入 claude 即可启动 Claude Code
