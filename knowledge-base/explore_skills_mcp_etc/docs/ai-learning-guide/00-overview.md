# AI 核心概念学习指南

> 面向有开发经验的 AI 工具使用者，系统学习 LLM、Agents、Skills、MCP 等核心概念

---

## 课程简介

本课程旨在帮助你理解现代 AI 工具背后的核心概念和工作原理。通过学习这些知识，你将能够：

- 理解 AI 工具为什么这样工作
- 更有效地与 AI 工具交互
- 做出更明智的技术选择
- 为深入学习打下坚实基础

## 适合人群

- 有一定编程经验
- 日常使用 AI 工具（如 ChatGPT、Claude、GitHub Copilot 等）
- 想要理解这些工具背后的原理
- 希望更高效地使用 AI 工具

## 课程结构

本课程分为 **6 大基础模块 + 5 大进阶模块**，共 **43 章**：

## 学习建议

### 学习路径

```
基础路径（推荐新手）：
模块一 → 模块二 → 模块三 → 模块四 → 模块五 → 模块六

快速了解路径：
第1章 → 第5章 → 第9章 → 第16章 → 第20章

按需学习路径：
根据你的兴趣选择模块深入学习
```

### 学习方法

1. **理解概念**：每个概念都配有生活化的类比，帮助你建立直觉
2. **动手实践**：理论与实践结合，每个模块都有练习
3. **循序渐进**：从基础到进阶，逐步深入
4. **联系实际**：将学到的知识与日常使用的 AI 工具对应起来

### 时间安排建议

- **快速浏览**：每个模块 30-60 分钟
- **深入学习**：每个模块 2-4 小时
- **完整学习 + 实践**：整体课程 20-30 小时
