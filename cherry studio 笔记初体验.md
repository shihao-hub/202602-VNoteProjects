# cherry studio 笔记初体验

1.  **“输入 / 调用命令”的提示太碍眼了**，不仅干扰操作，还显得多余。
    
2.  内置功能看似丰富，但大多体验一般。对我而言，真正实用的是它整合了多个小程序——省去了下载一堆独立 App 的麻烦。<br>可惜的是，**小程序最多只能同时开启三个**，这点真的很让人抓狂。
    
    -   目前我主要用的是 **Gemini、Qwen 和 Z.AI**，基本上够用了。
        
    -   其中，**Gemini 是付费的**（既然花了钱，不用就亏了），**Qwen 完全免费**，而 **Z.AI（智谱出品）在编程方面表现不错**。
        
3.  **不得不提的是**，这款工具最大的优势在于 **大幅减少了需要安装的软件数量**。<br>就目前使用情况来看，**至少帮我省下了四个独立应用**，这一点非常值得肯定！
    
4.  单纯的记笔记方便，平替 typora 和 vnote 了，vnote 当阅读器和编辑器都不错，typora 对我而言只适合当阅读器，cherry studio 的笔记对我而言完全替代了他们两个，**虽然有瑕疵，但是瑕不掩瑜**。
    
    -   其实我发现，拥有这种能力，**似乎完全因为伟大的浏览器技术**：跨平台、善复用… Electron + Browser…
