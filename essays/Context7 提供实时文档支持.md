> “Context7 将最新的、特定版本的文档和代码片段等信息，直接整合到你给大模型的提示（prompt）中……像给大模型配备了一个实时搜索引擎。” —— 来源：Context7 博客 & GitHub README

**Context7 是一个为 AI 编程助手（如 Cursor、Claude）提供最新技术文档的服务。**

它的工作原理是：

1.  **自动抓取** GitHub 和官方文档中的最新内容；
    
2.  **解析并切分**成带代码示例的小片段；
    
3.  **用向量索引**，支持语义搜索；
    
4.  当你在编辑器中提问时，通过 **MCP 协议** 把最相关的文档片段 **实时注入** 到 AI 的提示词中。
    

结果：AI 写出的代码基于**真实、最新、可运行的官方示例**，大幅减少“幻觉”。简单说：Context7 = 给 AI 配了一个会查最新文档的助手。
