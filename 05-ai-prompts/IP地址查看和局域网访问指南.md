# IP地址查看和局域网访问指南

## 一、如何查看当前电脑的IP地址

### 通过命令行查看
在命令行中运行：
```bash
ipconfig
```

### 查看公网IP地址
- **命令行方式**:
  ```bash
  curl ifconfig.me
  ```
  或
  ```bash
  curl ip.sb
  ```

- **浏览器访问**: 访问网站如 `ip.sb` 或 `ifconfig.me`

- **通过设置查看**:
  - 右键点击任务栏网络图标 → 打开"网络和Internet设置"
  - 点击"属性"查看网络详情

---

## 二、IPv4 地址和默认网关的区别

### IPv4 地址
- **例如**: `192.168.89.187`
- 这是**你的电脑**在局域网中的唯一标识
- 就像你家的"门牌号"
- 其他设备可以通过这个地址找到你的电脑

### 默认网关
- **例如**: `192.168.88.1`
- 这是**路由器**的地址
- 就像"小区出口"
- 负责把你电脑的数据转发到外网（互联网）

### 简单比喻
- 你的电脑 = 房间 → IP: 192.168.89.187
- 路由器 = 楼道出口 → 网关: 192.168.88.1

---

## 三、让局域网其他人访问你的服务

**你应该使用：IPv4 地址 `192.168.89.187`**

例如，如果你在电脑上运行了一个Web服务器（监听端口8080），局域网内其他人可以通过以下地址访问：

```
http://192.168.89.187:8080
```

### 注意事项

1. **防火墙设置**: 确保Windows防火墙允许该端口的入站连接
2. **端口**: 别忘了加上端口号（如果不是80端口）
3. **服务监听地址**: 确保你的服务监听在 `0.0.0.0`（所有接口）或者 `192.168.89.187`，而不是 `127.0.0.1`（只允许本机访问）
